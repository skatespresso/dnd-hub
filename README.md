# dnd-hub / Tavernia / Campaigner - Källkodsprojekt

## Beskrivning
Detta är en React-applikation som presenterar mitt projektarbete för kursen. 
Projektet består av en kampanj-hub för Dungeons & Dragons-spelare.

## Installation

### Förutsättningar
- Node.js (v18 eller senare)
- npm eller yarn

### Steg för att köra projektet
1. Klona repositoryt
2. Installera dependencies: `npm install`
3. Starta utvecklingsservern: `npm run dev`
4. Öppna webbläsaren på `http://localhost:5173`

## Teknisk stack
- React 18
- Vite (build tool)
- React Router (för navigation mellan vyer)
- Shadcn UI
- Tailwind
